# Vessel Client

## Client Requirements


Minimum recommended system requirements:

* Dual Core CPU
* 2GB RAM
* At least 200MB Disk Space
* 25 Mbps Upload Speed

Recommended system requirements:

* Quad Core CPU
* 8GB RAM
* At least 1GB Disk Space
* 50 Mbps Upload Speed

Supported Operating Systems

* Ubuntu Linux
* Windows 7/10
