# Building

## Supported Platforms

* Windows 7/10
* Ubuntu Linux

## Requirements

* CMake
  * Download: https://cmake.org/download/
* C++ Compiler
  * GCC/G++ (Recommended)
  * Clang

## Building the Client Application

### Required Libraries

* libdl (Linux)
* pthreads
* LibArchive
* Boost
  * system
  * filesystem
  * date_time
  * thread
  * log
  * log_setup
  * program_options
* sqlite3
* OpenSSL
* CryptoPP
